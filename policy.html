クチコミ情報の「健全性」と真摯に向き合います

PILOT VALUE に掲載されている情報は、あくまでも投稿者であるパイロット個人の体験に基づく主観的な意見・感想です。原則として、掲載内容の正確性、最新性、有益性など、あらゆる点に関して PILOT VALUE が内容を完全に保証できるものではございません。プロフェッショナル同士の参考情報の一つとしてご活用ください。

しかしながら、航空業界という高い公共性と責任を伴うコミュニティを運営する上で、情報の健全性に真摯に向き合い、日々改善・改良を行っていくことが、私たち PILOT VALUE 運営者の責任であると考えています。

クチコミのクオリティ維持・向上
運営スタッフによる専門的目視審査

PILOT VALUE では、クチコミ情報を共有するプラットフォームの責任として、航空業界および社会に対して不利益やネガティブな影響を与えないよう、最大限の努力を払っています。

投稿されたレポートに対しては、運営スタッフが目視での審査を実施。独自のレポート回答ガイドラインに基づき、機密情報や不適切な表現がないかを厳査し、不適切と判断した場合には非掲載の処置をとっています。

レポート回答時の厳格なルール

「情報の質」を担保するための取り組みとして、レポートの回答には 200文字以上 という条件を設定し、投稿のハードルを設けています。また、コピー＆ペースト機能を制限することで、安易な転載を排し、個々のパイロットが実体験に基づいた「血の通った情報」のみが集まる仕組みを構築しています。

不適切なクチコミの報告（ピア・レビュー）システム

情報の閲覧ページには、読み手が不適切・不正確と感じた投稿に対して、運営に報告できるフォームを設置しています。報告を受けた際は速やかにスタッフが内容を確認します。また、有益な不適切通知を行ったユーザーには、サイト内で利用可能なポイントを付与するなど、コミュニティ全体で情報の精度を高める施策を行っています。

会社評価スコアの算出について

PILOT VALUE の評価スコアは、実際に勤務経験のあるパイロットから回答された有効な評価の集計です。点数は単純平均ではなく、より信頼性の高い「市場価値」を示す指標とするために、独自のアルゴリズムにより算出しています。また、特定の時期の偏りを防ぐため、一定の回答数が集まるまでスコアが過度に変動しないロジックを採用しています。

より有益な情報を届けるために重視している3つのポイント

  1. 1. 最新性：航空業界の待遇変化は激しいため、直近の情報をより重く反映します。
  2. 2. 回答者数：統計的な信頼性を担保するため、母数に応じたスコアリングを行います。
  3. 3. 乖離性：極端に偏った評価（悪意ある攻撃や過度な賞賛）を検知し、スコアへの影響を調整します。
航空会社にとっても良きパートナーとなる

PILOT VALUE は、中立的な立場から、広告ではない「生の声」を通じて各航空会社の真の魅力を伝える役割を担いたいと考えています。

「パイロットを大切にする企業」が「そうでない企業」よりも正当に評価され、誠実な経営を行う航空会社にこそ、最高のプロフェッショナルが集まる環境を創出すること。それが、2030年問題を控えた航空業界全体の健全な発展に貢献すると私たちは信じています。
