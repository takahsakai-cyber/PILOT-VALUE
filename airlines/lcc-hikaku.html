DEEP DIVE

なぜLCCはANA・JALより安いのか？

単なる「安い航空会社」だからではない。構造的な理由が3つある。

✈️

① 単一機材＝訓練費削減

LCCは1〜2機種のみ運航。パイロットの機種転換訓練が不要で人件費を低く抑えられる。一方、大手は多機種対応が必要で訓練コストが高く、それが給与に反映される。

🔄

② 高回転運航モデル

短距離・高頻度の折り返し運航が基本。パイロット1人あたりの飛行時間は多いが、1フライトの単価が低い。フライト単価×時間で給与計算するLCCは、大手の長距離運航より収益効率が落ちる。

📊

③ 利益率の構造的限界

LCCの利益率は大手より薄い。座席単価を下げてボリュームで稼ぐモデルのため、人件費に回せる原資が少ない。特に日本のLCCは市場競争が激しく、価格訴求を優先せざるを得ない。

SKYMARK DEEP DIVE

Skymarkだけなぜ高い？

Skymarkの推定年収¥2,900万はLCCの中で突出している。その理由が興味深い。

Skymarkが高い3つの理由

1
独立系のため給与体系が自由
ANA・JALグループのLCCは親会社の給与テーブルに縛られる面がある。独立系のSkymarkは優秀なパイロットを引き付けるため給与を高めに設定できる。
2
2015年民事再生後の経営改善
経営危機を経て人材確保の重要性を学んだSkymarkは、パイロット待遇改善に積極的。特にベテランパイロットの引き留めに注力している。
3
B737特化で効率的な訓練体制
B737一機種に特化することで訓練コストを抑えながら、その分を給与に回す戦略。

Skymarkに向く人・向かない人

向く人
  • • 羽田ベースで働きたい
  • • 国内線中心でWLBを重視
  • • 早期昇格で機長になりたい
  • • LCCの中で最高年収を狙いたい
向かない人
  • • 国際線・長距離便を飛びたい
  • • 多機種の経験を積みたい
  • • ANA・JAL並みの年収を目指す
  • • 大企業の安定感が必要
ZIPAIR DEEP DIVE

ZIPAIRは「特殊なLCC」——B787で長距離を飛ぶ

他のLCCと根本的に違う点

✓
B787運航——国内LCCで唯一のワイドボディ機。機種資格の価値が高い。
✓
JAL子会社——将来的にJALへの転籍・出向の可能性がゼロではない（確約なし）。
✓
中長距離国際線——東アジア・北米・東南アジアへのフライト経験が積める。
✓
2020年設立の新興キャリア——組織が若く、成長過程で昇格チャンスがある。

ZIPAIRの注意点

長距離LCCの拘束時間
国際線長距離便は拘束時間が長く、WLBはSkymark・Peachより悪い。
まだ規模が小さい
設立4年でまだ路線数・機材数が少なく、安定性はLCC最低水準。
成田ベース固定
現状は成田のみ。地方在住者にとっては通勤負担が大きい。
CAREER GUIDE

LCC vs 大手——どちらを選ぶべきか？

「年収が低いからLCCは選ばない」は間違い。キャリアフェーズによって正解が変わる。

LCCを選ぶべき人

✓
早期機長昇格を重視する人

大手で8〜12年待つより、LCCで3〜5年で機長になる選択肢。若い機長資格はエミレーツ等への転職でも評価される。

✓
国内線でWLBを確保したい人

家族との時間を優先したい場合、国内線専業のPeach・Skymarkは海外勤務なしで安定した生活が可能。

✓
ANA・JALに入れなかった・やめた人

大手から退職後の再就職先として。LCCは経験年数より現在の飛行技術を重視する傾向あり。

大手（ANA・JAL）を選ぶべき人

✓
長期的な年収最大化を目指す人

大手機長の最大年収4,200万円はLCCの約1.5倍。昇格が遅くても、40代以降の累計収入では大手が上回る。

✓
多機種・長距離の経験を積みたい人

B777・A380・B787など多機種の経験は、将来の海外転職（エミレーツ・カタール等）で高評価。LCCの単一機種では代替できない。

✓
外資系転職を見据えている人

エミレーツ・カタールの採用条件には「広体機での機長経験」を求めるケースが多い。Peach・SkymarkのA320/B737では条件を満たせない場合がある。

FAQ

よくある質問

LCCパイロットからANA・JALに転職できますか？+
難易度は高め。ANA・JALは基本的に自社養成か航大出身者を優遇。ただし近年のパイロット不足で中途採用が増えており、LCC機長経験者の採用事例も出ている。飛行時間と評価によっては可能性はある。
LCCパイロットはエミレーツに転職できますか？+
エミレーツの採用要件は「広体機（ワイドボディ）での機長経験3,500時間以上」が基本条件。Peach（A320）・Skymark（B737）・Jetstar Japan（A320）は狭胴機のため、原則この条件を満たさない。ZIPAIRのB787機長経験であれば可能性あり。
LCCのパイロットは休みが取れますか？+
国内線専業LCC（Peach・Skymark）は宿泊を伴うフライトが少なく、拘束時間はANA・JALより短い傾向。ただし折り返し便が多く1日のフライト本数は多め。「長い海外ステイがない代わりに毎日飛ぶ」イメージ。ZIPAIR（国際線）は大手と同様に長距離拘束あり。
PeachはANAの子会社なのになぜ給与が低いですか？+
Peachは法人として独立した会社であり、ANA本体とは別の給与体系。親会社のブランドを活用しながらコストを抑えるためにLCCとして分離されている構造上、給与水準も分離されている。同様にJetstar JapanもJALグループとは別給与。
将来性はLCCと大手どちらが高いですか？+
航空市場の長期トレンドはLCC拡大。2030年代に向けてアジア太平洋のLCC旅客数は大手を上回ると予測されている。ただしパイロットとしての「価値」はLCCより大手・外資系の方が高い傾向。「会社の成長性」と「個人の年収・キャリア」は別軸で考える必要がある。
